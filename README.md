## Краткая теория (вводная часть)

### Что такое REST?

**REST** (Representational State Transfer) — это архитектурный стиль для построения
веб-API. REST — не протокол и не стандарт, а набор принципов, которым должен
следовать API, чтобы называться **RESTful**.
